| **Overview** | ***In breve*** |
| The O64ml Project allows you to develop OCaml programs and compile them on Linux systems to produce code executable on a Commodore 64. The Project includes, in addition to the `o64ml` compiler, a specially modified distribution of the OCaml Standard Library and (coming soon) libraries dedicated to graphics and sound. | *Il Progetto O64ml consente di sviluppare programmi in OCaml e compilarli su sistemi Linux producendo codice eseguibile da un Commodore 64. Il Progetto comprende, oltre al compilatore `o64ml`, una distribuzione della Standard Library di OCaml appositamente modificata e (prossimamente) delle librerie dedicate alla grafica e al suono.* |
